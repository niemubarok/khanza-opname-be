# khanza-opname-be
backend untuk aplikasi stok opname khanza


## 1. Download node js versi genap 
      https://nodejs.org/en/download/
      
## 2. Install dependencies
      npm install

## 3. Edit .env

## 4. Jalankan Migrasi 
      node ace migration:run
      
## 5. Jalankan server
      node ace serve --watch
